# 🌦️ Weather App

## 📌 Description
This is a simple Weather App that shows real-time weather data using API.

## 🚀 Features
- Search weather by city name
- Shows temperature, humidity, wind speed
- Simple and clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🔑 API Used
This project uses OpenWeatherMap API to fetch real-time weather data.
Website: https://openweathermap.org/api
You need to add your own API key in the Javascript file

## ▶️ How to Run

1. Clone this repository  
   git clone https://github.com/VIVEKSEN199/weather-app.git  

2. Open the project folder  

3. Open index.html in your browser  

4. Enter city name to get weather details
